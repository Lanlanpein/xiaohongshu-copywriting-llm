# 爆款小红书文案生成器（Web版）｜5个标题 + 1篇完整推文

> 关键词：AI大模型应用｜Python｜Web应用｜Prompt工程｜内容生成｜新媒体运营  
> 特点：本地运行网页，输入 API Key + 主题即可生成爆款文案

本项目是我独立自学完成的 AI 大模型应用 Demo。  
通过 Python 在本地运行一个网页应用，用户输入自己的大模型 API Key 后，
只需要填写“小红书主题/产品信息”，即可自动生成：

✅ 5 个爆款标题（多风格）  
✅ 1 篇完整种草推文（结构化输出）  

适用于：自媒体运营、品牌种草、电商推广、内容提效。
